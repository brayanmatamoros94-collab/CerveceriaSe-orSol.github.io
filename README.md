# Cerveceria SeñorSol.github.io
🍺  ¡Bienvenido a SeñorSol Cerveceria! -  Catálogo de Cerveza Artesanal ,  Este es el sitio web oficial de nuestra microcervecería, donde la tradición se encuentra con la innovación. Aquí mostramos nuestra selección de temporada, el proceso de elaboración y nuestra historia.

🚀 Características del Proyecto

    Catálogo Interactivo: Galería visual de nuestras variedades (IPA, Stout, Lager).

    Localizador de Puntos de Venta: Mapa integrado para encontrar tu bar más cercano.

    Responsive Design: Optimizado para móviles (porque la sed ataca en cualquier lugar).

    Blog Cervecero: Artículos sobre el proceso de fermentación y maridaje.

🛠️ Tecnologías Utilizadas

Este sitio fue construido con:

    HTML5 & CSS3 (Flexbox y CSS Grid para el diseño).

    JavaScript (ES6) para las animaciones y filtros del catálogo.

    Google Maps API para el localizador.
